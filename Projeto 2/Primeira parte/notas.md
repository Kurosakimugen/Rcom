## URL structure

    <protocol>://<user>:<password>@<host>:<port>/<url-path>

-
    
    193.136.28.10
    
    <URL:  ftp://@host.com/> has an empty
    user name and no password, <URL: ftp://host.com/> has no user name,
    while <URL:ftp://foo:@host.com/> has a user name of "foo" and an
    empty password.
